# crs-ivr
Asterisk IVR interface for Paging Control Stack

## Preparing a local repository

1. `su`
1. `yum -y install git`
1. `mkdir crs-asterisk`
1. `cd crs-asterisk`
1. `git init`
1. `git remote add origin https://github.com/criticalresponse/crs-asterisk.git`
1. `git pull origin master`

## Build and Install
1. `su`
1. `./setup-16`
1. `./build1-16`
1. `./build2-16`
